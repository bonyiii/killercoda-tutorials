| Koncepció       | Hasonlat                          |
|-----------------|-----------------------------------|
| Objektum        | Épuletanyag (tégla, csavar)       |
| Osztály         | Tervrajz                          |
| Példány         | Valódi ház                        |
| Öröklés         | IS-A (Ház IS-A Épulet)            |
| Mixin           | HAS-A (eloregyártott felszerelés) |
| Enkapszuláció   | Falak és magánszféra              |
| Polimorfizmus   | Sok szoba, egyforma használat     |
| Duck Typing     | "Kacsaként viselkedik"            |
| Metódus keresés | Elektromos vezetékek követése     |
